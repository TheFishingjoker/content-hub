Skyvern is an AI-powered browser automation tool built on Vision LLMs. Unlike traditional browser automation that relies on DOM parsing and XPath selectors (which break when layouts change), Skyvern uses visual AI to interact with websites in a human-like way — adapting to layout changes automatically. Inspired by Task-Driven autonomous agent design (BabyAGI, AutoGPT), uses Playwright under the hood.
