    17|Open-source memory layer for AI agents. Entity-centric model where users and agents are "peers" — Honcho builds psychological models of each peer from conversation history via an async reasoning pipeline. Three specialized agents work together: Deriver (memory formation), Dialectic (analysis/recall via natural language queries), Dreamer (memory consolidation). Supports multi-participant sessions, vector search, RAG via Collections/Documents. FastAPI + PostgreSQL + pgvector. SDKs on PyPI (honcho-ai) and npm (@honcho-ai/sdk). 2.5K stars, AGPL-3.0, by Plastic Labs.
