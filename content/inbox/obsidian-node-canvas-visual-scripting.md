    17|Visual scripting engine that turns Obsidian Canvas into a workflow automation platform. Compose function blocks as nodes on a canvas, connect them with arrows, then execute the graph via CLI. Supports LLM integration (OpenAI/LangChain), browser automation (Puppeteer), REST APIs, templated file writing (YAML + Nunjucks), and complex flow control — aggregation, zip-merging, branching. Built with TypeScript, runs on Node.js 20+.
    18|
    19|Key deps: openai, langchain, puppeteer, playwright, cheerio, nunjucks, zod. Prototyped at a Berlin hackathon. 15 stars, TypeScript.
