Open-source financial data platform. Unified API for equity, crypto, options, fixed income, derivatives, and economics data. "Connect once, consume everywhere" — exposes data to Python, Excel, MCP servers for AI agents, and REST APIs. 66K stars, AGPL-3.0, Python. Integrates with OpenBB Workspace. Could be useful as a data layer for Polymarket research — pulls from multiple financial providers into one interface.
