# XyOps

Crontab replacement that doubles as a server monitor and ticketing system. Tracks CPU/RAM per job and auto-creates tickets with full server snapshots on crash.

**Key facts:**
- Replaces crontab with monitoring built in
- Tracks CPU and RAM per scheduled job
- Auto-creates tickets with server state snapshots on failure
