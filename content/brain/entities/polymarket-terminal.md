# Polymarket Terminal

Automated Polymarket trading bot. Three strategies: Maker Rebate MM (market-neutral spread capture), Copy Trader (wallet mirroring), Orderbook Sniper (deep-discount limits). Node.js, CLOB API, Polygon RPC, WebSocket.

**Key facts:**
- 230 stars, MIT, JavaScript/Node.js 18+
- Maker Rebate MM: simultaneous YES/NO limits, CTF merge for profit
- Copy Trader: proportional sizing, auto-sell, auto-redeem
- Orderbook Sniper: 3-tier GTC limits, time-based sizing
- Uses RTDS WebSocket for fill detection
- PM2 for VPS process management
