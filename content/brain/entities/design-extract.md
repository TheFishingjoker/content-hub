# design-extract

CLI tool that crawls a live website's DOM and reverse-engineers its complete design language: colors, fonts, grid patterns, hover transitions, responsive breakpoints.

**Key facts:**
- Outputs: Tailwind config, CSS variables, shadcn/ui theme, Figma variables
- Can be installed as a Claude Code skill
- Usage: /extract-design stripe.com → replicates the design system
