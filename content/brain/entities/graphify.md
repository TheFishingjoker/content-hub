# Graphify

Reads code, Markdown, PDFs, screenshots. Outputs interactive visual graph, Obsidian vault, Wikipedia-style wiki, and plain-English report. 71.5x fewer tokens per query than raw files. Runs locally, no Neo4j, no server.

**Key facts:**
- Local-first, no server needed
- Generates Obsidian vault + wiki from raw files
- Semantic graph of your knowledge
