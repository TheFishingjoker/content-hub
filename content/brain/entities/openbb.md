# OpenBB

Open-source financial data platform. Unified API for equity, crypto, options, fixed income, derivatives, economics. 66K stars, AGPL-3.0, Python/FastAPI. MCP server support for AI agents.

**Key facts:**
- 66K stars, AGPL-3.0, Python 3.9-3.12
- Unified API: equity, crypto, options, fixed income, derivatives, economics
- FastAPI backend, PyPI package (pip install openbb)
- MCP server support for AI agent integrations
- Integrates with OpenBB Workspace and Excel
- Extensive data provider integrations
- Could serve as data layer for financial/polymarket research
