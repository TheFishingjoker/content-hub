    11|# obsidian-node-canvas
    12|
    13|Visual scripting engine built on Obsidian Canvas. Reads `.canvas` JSON files and executes node graphs as workflows — LLM calls, web scraping, file writing, flow control. TypeScript, Node.js 20+, 15 stars.
    14|

    16|- Turns Obsidian Canvas into visual programming environment
    17|- Node types: start, code (JS/TS), YAML, decision, identity
    18|- Supports LLM (OpenAI/LangChain), Puppeteer, Cheerio, Nunjucks templating
    19|- Flow control: aggregation, zip-merging, branching
    20|- 15 stars, TypeScript, prototyping stage
    21|- Dependencies: openai, langchain, puppeteer, playwright, cheerio, nunjucks, zod
