# Skyvern

AI-powered browser automation tool built on Vision LLMs. Replaces brittle XPath/DOM-based automation with visual AI that interacts with websites like a human.

**Key facts:**
- Uses Vision LLMs to understand page layouts visually
- Adapt to layout changes automatically (no broken selectors)
- Built on Playwright for browser interaction
- Inspired by Task-Driven agents (BabyAGI, AutoGPT style)
