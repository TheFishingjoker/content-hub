    11|# Honcho
    12|
    13|Memory infrastructure for AI agents. Builds entity-centric psychological models from conversation history. Three internal agents: Deriver (memory formation), Dialectic (natural language recall), Dreamer (consolidation). FastAPI + PostgreSQL + pgvector.
    14|
    15|**Key facts:**
    16|- 2.5K stars, AGPL-3.0, by Plastic Labs
    17|- Entity-centric "peer" model — users and agents both represented as peers
    18|- Async reasoning pipeline builds psychological models over time
    19|- Multi-participant sessions with configurable observation
    20|- Vector search + RAG via Collections/Documents
    21|- Python (core), TypeScript (SDK), FastAPI, PostgreSQL+pgvector, Redis
    22|- LLM providers: Anthropic, OpenAI, Gemini, Groq
    23|- PyPI: honcho-ai, npm: @honcho-ai/sdk
